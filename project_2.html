Project-2: In-Depth Bulk RNAseq Data Analysis

Overview

This project focuses on analyzing bulk RNA sequencing (RNAseq) data to identify differentially expressed genes, pathways, and potential biomarkers in colorectal cancer. By understanding the transcriptional landscape of cancerous tissues compared to normal tissues, we aim to uncover insights that can inform therapeutic strategies and biomarker discovery.

Objectives

  • Identify differentially expressed genes between tumor and normal samples.
  • Perform pathway enrichment analysis to identify dysregulated pathways.
  • Construct gene co-expression networks.
  • Visualize the results using interactive plots.

Methodology

  1. Data Collection

    Retrieve bulk RNAseq data from public repositories such as TCGA for colorectal cancer.

  2. Data Preprocessing

    Preprocess the raw RNAseq data, including quality control, normalization, and filtering.

  3. Differential Expression Analysis

    Use statistical methods to identify differentially expressed genes between tumor and normal samples.

  4. Pathway Enrichment Analysis

    Perform pathway enrichment analysis to identify significant biological pathways associated with the differentially expressed genes.

  5. Gene Co-Expression Network Analysis

    Construct and analyze gene co-expression networks to identify key regulatory genes and modules.

  6. Visualization

    Create interactive visualizations to represent the analysis results.

Results

The results of this project include:

  • Lists of differentially expressed genes.
  • Enriched pathways and their associated genes.
  • Gene co-expression networks and key modules.
  • Interactive visualizations of the analysis results.

Interactive Elements

  • Differential Expression Heatmap: Interactive heatmap showing differentially expressed genes.
  • Pathway Enrichment Plot: Interactive plot of enriched pathways.
  • Co-Expression Network: Interactive gene co-expression network visualization.
